## Guía de estilo para TODOS los prompts

Incluí estas instrucciones en cada prompt o como contexto de sesión en Gemini:

```
Estilo visual: Dark, cinematográfico, monocromático. Fondo negro (#0D0D0D).
Paleta: Solo blanco y negro. Sin color excepto cuando se indique explícitamente.
Mood: Estadio vacío a medianoche. Misterioso. Streetwear underground.
Iluminación: Dramática, lateral, con sombras profundas. Luz dura tipo spot.
NO incluir texto en la imagen. El texto se agrega después.
Aspect ratio: 9:16 para stories, 1:1 para feed.
```

## TIPS PARA GEMINI

1. **Si la imagen no sale perfecta:** Ajustá el prompt, no te conformes. Regenerá.
2. **Consistencia:** Hacé todas las de un mismo día en una sesión para mantener estilo.
3. **Post-producción:** El texto SIEMPRE se agrega después (IG editor o Canva). Gemini no es bueno con texto.
4. **Aspecto:** Verificá que cada imagen sea 9:16 (stories) o 1:1 (feed) antes de usarla.
5. **Si Gemini genera algo con color no deseado:** Agregá "strictly monochrome, black and white only, no color" al prompt.
6. **Guardar todo en:** `el-club/assets/img/campaign/` para tener backup.
